## 用自己的话回答

- 什么是特征，什么是标签：
- 特征是模型用来做出预测的输入信息，比如用户的月消费金额、登录频次、套餐类型这类可观测的属性；标签是我们希望模型最终预测得出的目标结果，比如用户会不会流失这类需要预判的答案。
- 为什么要保留测试集：
- 测试集全程不参与模型训练过程，用来模拟模型从没见过的新数据，检验模型在陌生场景下的泛化能力，避免模型只死记训练数据出现过拟合，以此得到更真实可靠的模型效果评估结果。
- 为什么83%准确率仍可能没有用：
- 如果数据集类别分布极不均衡（比如仅 17% 用户会流失、83% 用户留存），模型只要无脑全部预测 “不流失” 就能拿到 83% 准确率，但它完全识别不出会流失的用户，没法实现业务挽留的目标，在实际业务里就没有实用价值。
